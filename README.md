## 📂 Featured Project

### 📊 Sales & Profit Analysis Dashboard

**Tools Used:** SQL, Excel, Power BI

This project explores sales and profitability trends to understand:
- Product performance
- Regional revenue contribution
- Time-based sales trends

---

### 🔍 Key Insights

- Technology products generated the highest sales and profit  
- Furniture showed lower profitability despite consistent sales  
- Sales peaked toward the end of the year (Q4 trend)  
- Certain regions contributed a disproportionately high share of revenue
